Simple database for our little project

Please understand that this is just a really simple stuff and may not contain every goddamn function and feature that you may crave.

What it does

You can add items to storage (if you have some) so we can all know who added what, where, when and how many.

What it does not do

There is no login (so far) so please be careful with who you are sharing this link: akela.mendelu.cz/~xcizek/ME. Othervise we might get some unpleasant spam in our database and it would be a real pain to recover what we had before that.

So long as there is no Authentication, there is no warranty that it has correct data.
